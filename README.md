# Installation

Clone the repository:
```bash
git clone https://github.com/S0meR4nd0mGuy/Custom-CLiParse-and-Logging.git
```

Then install whichever module(s) you need globally using pip:
```bash
pip install ./Custom-CLiParse-and-Logging/cliparse
pip install ./Custom-CLiParse-and-Logging/advanced_logging
```

Or if you're already inside the cloned folder:
```bash
pip install ./cliparse
pip install ./advanced_logging
```
